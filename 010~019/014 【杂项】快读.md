# 这只是一个快读的板子

```cpp
inline int read(void)
{
    char ch = getchar();
    int num = 0, sign = 1;
    while (ch != EOF && (ch < '0' || ch > '9')) 
    {
        if (ch == '-') sign = -1;
        ch = getchar();
    }
    while (ch != EOF && ch >= '0' && ch <= '9')
    {
        num = num * 10 + ch - '0';
        ch = getchar();
    }
    return num * sign;
}
```
